@short: defines the position of a label of a combobox

@signature: {'labelPosition?: "left" | "top";'}

@example:
var combo = new dhx.Combobox("combo_container", {
    labelPosition:"left"
});

@descr:

**Related sample**: [Combobox. Label Position](https://snippet.dhtmlx.com/2936fray)

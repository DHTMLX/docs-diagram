title: getFocus()
---          

@short: returns the id of an item in focus

@signature: {'getFocus(): string | number;'}

@returns:
A string, or number with the id of an item in focus.

@example:
dataview.getFocus(); // -> "item_id"

@changelog: added in v6.3

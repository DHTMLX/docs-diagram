@short: sets the width of a tab

@signature: {'tabWidth?: number | string;'}

@default: 200

@example:
var tabbar = new dhx.Tabbar("tabbar_container", {
    tabWidth: 120
});

@descr:

**Related sample**: [Tabbar. Tab Size](https://snippet.dhtmlx.com/yy841z3j)

@changelog:

The "string" type is added in v7.0.

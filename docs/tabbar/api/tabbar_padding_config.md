sidebar_label: padding
title: padding
---          

@short: defines the distance between the content of a cell and the border of tabbar

@signature: {'padding?: number | string;'}

@example:
var tabbar = new dhx.Tabbar("tabbar_container",{
	views:[ 
    	{ tab: "left", padding:20},
    	{ tab: "west", padding:20},
    	{ tab: "east", padding:20},
    	{ tab: "right", padding:20}
	]
});

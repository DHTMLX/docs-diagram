@short: gets the ID of the parent of the control

@signature: {'getParent(id: Id): Id;'}

@params:
`id: string` - the ID of a control

@returns:
A string with the ID of the parent of the control.

@example:
var parentId = toolbar.data.getParent("add_btn");

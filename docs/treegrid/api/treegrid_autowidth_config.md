# autoWidth

@short: makes treegrid's columns to fit the size of a treegrid

@signature: {'autoWidth?: boolean;'}

@default: false

@example:
var treegrid = new dhx.TreeGrid("treegrid_container", {
    columns: [// columns config],
    autoWidth:true, /*!*/
    data: dataset
});

@descr:

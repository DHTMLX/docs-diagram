@short: returns an array with IDs of selected items

@signature: {'getSelected(): string[];'}

@returns:
An array of string values with IDs of selected items.

@example:
toolbar.getSelected(); // -> ["selected_1", "selected_1.1"]

@descr:

@changelog:
added in v7.0

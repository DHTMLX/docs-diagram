@short: adds an icon with a question mark next to Slider

@signature: helpMessage?: string;

@example: 
var slider = new dhx.Slider("slider_container",{
	helpMessage:"Some text"
});



@template:	api_config
@descr: 
The property is useful while using Slider in form.

---          

@short: defines whether ColorPicker is shown only in the picker mode

@signature: {'pickerOnly?: boolean;'}

@default: false

@example:
var colorpicker = new dhx.Colorpicker("colorpicker_container",{
	pickerOnly: true
});

@descr:

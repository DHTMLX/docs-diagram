@short: returns the code of a selected color

@signature: {'getValue(): string;'}

@returns:
- value    string   the code of a selected color in the Hex format

@example:
colorpicker.setValue("#BDF0E9");

var color = colorpicker.getValue();
// -> "#BDF0E9"

@short: shows a section with custom colors in the bottom part of ColorPicker

@signature: {'customColors?: string[];'}

@example: 
var colorpicker = new dhx.Colorpicker("colorpicker_container", {
	customColors: ["#1d2de7", "#ab31ff", "#a3fa76"]					
});

@descr:

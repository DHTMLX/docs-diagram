# autoWidth

@short: makes grid's columns to fit the size of a grid

@signature: {'autoWidth?: boolean;'}

@default: false

@example:
var grid = new dhx.Grid("grid_container", {
    columns: [// columns config],
    autoWidth:true, /*!*/
    data: dataset
});

@descr:

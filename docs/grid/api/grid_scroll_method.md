---          

@short: scrolls a grid according to specified coordinates

@signature: {'scroll(x?: number, y?: number): void;'}

@params:
- `x: number` - the X coordinate
- `y: number` - the Y coordinate

@example:
grid.scroll(75,230);

@descr:

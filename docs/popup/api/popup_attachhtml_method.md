@short: adds an HTML content into a dhtmlxPopup

@signature: attachHTML(html: string): void;

@params:
- html		string		an HTML content to be added into a popup

@example:
var popup = new dhx.Popup();
popup.attachHTML("<h1>Hello, I'm popup</h1>");
popup.show("popup");

@short: sets the position of a window

@signature: {'setPosition(left: number, top: number): void;'}

@params:
- left	number		the left coordinate of a window
- top	number		the top coordinate of a window

@example:
dhxWindow.setPosition(20,20);
dhxWindow.show();

@descr:

---          

@short: checks whether a Checkbox control is checked

@signature: {'isChecked(): boolean;'}

@returns:
`true`, if a control is checked. Otherwise, `false`.

@example:
form.getItem("checkbox").isChecked(); // -> true/false

@descr:

@changelog: added in v7.0

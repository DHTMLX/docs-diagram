# disable()

### Description

@short: blocks HistoryManager

### Usage

~~~js
disable(): void;
~~~

The Reset and HistoryManager buttons from the Toolbar work only in the custom event generation mode. **Ctrl+z** and **Ctrl+Shift+z** navigation is not available.

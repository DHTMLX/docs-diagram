# getNearId()

@short: looks for a nearest item relative to the specified one

@signature: {`getNearId(id: Id): Id;`}

@params:

- `id: string | number` - the id of an item

@returns:
The id of the nearest item.

@example:
var nearId = diagram.data.getNearId(id);

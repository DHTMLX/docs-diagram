# getIndex()

@short: returns the index of the item by its id

@signature: {`getIndex(id: Id): number;`}

@params:
`id: string` - the id of the item

@returns:
The index of the item.

@example:
var index = diagram.data.getIndex("1"); // -> returns 0

@descr:

# getRoots()

### Description

@short: returns an array with ids of all root items of the diagram

@descr:

### Usage

~~~js
getRoots(): Id[]
~~~

### Returns

An array with the ids of the root items.

### Example

~~~js
diagram.data.getRoots();
~~~

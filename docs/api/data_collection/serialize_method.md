# serialize()

### Description

@short: serializes the diagram data into an array of JSON objects

@descr:

### Usage

~~~js
serialize(): T[];
~~~

### Returns

An array of JSON objects for each item and link from Diagram.

### Example

~~~js
const data = diagram.data.serialize();
~~~

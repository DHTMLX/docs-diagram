# getSubId()

### Description

@short: returns the subId (if any) of the selected item

### Usage

~~~js
getSubId(): Id;
~~~

### Returns


The subId (if any) of the selected item.

### Example

~~~js
const subid = diagram.selection.getSubId(); // -> "2"
~~~

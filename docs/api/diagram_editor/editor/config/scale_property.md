# scale

### Description

@short: Optional. Defines the diagram editor scale

### Usage

~~~jsx
scale?: number;
~~~

### Default config

~~~jsx
scale: 1
~~~

### Example

~~~jsx
const editor = new dhx.DiagramEditor("editor_container", {
    scale: 0.9
});
~~~

# shapeBarWidth

@short: sets the width of the left panel of the editor

@type: number, string

@default: 295px

@example:
const editor = new dhx.DiagramEditor(document.body, {
    shapeBarWidth: 190
});

@descr:
{{note The property is available only in the default mode of the editor (*type:"default"*).}}

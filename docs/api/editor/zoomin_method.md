# zoomIn()

@short: zooms in the diagram

@signature: {`zoomIn(step?: number): void;`}

@params:
`step: number` - the step of zooming in the diagram; *by default: 0.05*

@example:
editor.zoomIn(0.1);

@descr:

@changelog:
Added in v4.1.

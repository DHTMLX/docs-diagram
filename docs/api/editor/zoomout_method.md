# zoomOut()

@short: zooms out the diagram

@signature: {`zoomOut(step?: number): void;`}

@params:
`step: number` - the step of zooming out the diagram; *by default 0.05*

@example:
editor.zoomOut(0.1);

@descr:

@changelog:
Added in v4.1.

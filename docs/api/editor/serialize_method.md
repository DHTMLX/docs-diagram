# serialize()

### Description

@short: serializes the data of the editor into an array of JSON objects

@descr:

### Usage

~~~js
serialize(): IDataItem[];
~~~

### Returns

An array of JSON objects for each item from the editor data.

### Example

~~~js
const data = editor.serialize();
~~~

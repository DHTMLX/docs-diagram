# editMode

### Description

@short: Optional, hides/shows the left/right panels of the editor

### Usage

~~~js
editMode?: boolean;
~~~

### Example

~~~js
const editor = new dhx.DiagramEditor("editor_container", {
    editMode: false
});
~~~

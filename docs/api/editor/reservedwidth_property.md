# reservedWidth

### Description

@short: optional, defines the left offset for the diagram

### Usage

~~~js
reservedWidth?: number;
~~~

### Example

~~~js
const editor = new dhx.DiagramEditor("editor_container", {
    reservedWidth: 150
});
~~~
